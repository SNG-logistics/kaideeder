🚀 KAIDEEDER Deploy Helper

Plesk → Node.js → Run Node.js commands — เลือก npm แล้วก๊อปคำสั่งด้านล่างวางในช่อง

📥 ขั้นตอนที่ 1 — ก่อน build (ถ้า code เปลี่ยน)
Git
ใช้ Plesk → Git → Pull updates

⚠️ ดึงโค้ดใหม่จาก GitHub ก่อนทุกครั้ง

📦 ขั้นตอนที่ 2 — Install packages
npm
install --omit=dev

ติดตั้ง package ที่เพิ่มมาใหม่ (ไม่รวม devDependencies)

🚀 ขั้นตอนที่ 3 — Deploy คำสั่งเดียว!
npm
run deploy

✅ ทำทุกอย่างใน 1 คำสั่ง: prisma generate → next build → copy static → db push

🔧 คำสั่งแยก (ถ้าต้องการรันทีละขั้น)
npm
run db:generate

Generate Prisma client (v5)

npm
run build

Build Next.js (ใช้เวลา 3-5 นาที)

npm
run copy-static

Copy static files → standalone folder

npm
run db:push:prod

Push Prisma schema → production DB

ℹ️ หมายเหตุ
  • Plesk Node.js panel: เลือก npm จาก dropdown แล้ววางคำสั่งในช่อง (ไม่ต้องพิมพ์ "npm " นำหน้า)
  • ถ้า error prisma not found → รัน install --omit=dev ก่อน
  • ถ้า error 502 Bad Gateway → restart app ใน Plesk Node.js panel
